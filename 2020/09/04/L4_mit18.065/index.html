this note is only a list of the key points

eigenvectors and eigenvalues

Def
If Ax=λxAx=\lambda xAx=λx, then xxx is the eigenvector and λ\lambdaλ is the eigenvalue. Normally, with n×nn\times nn×n AAA, there will be nnn different eigenvectors and its paired eigen values.

    i.e. dimensionality reduction
    i.e. the eigenvector xxx is in the nullspace and AAA is not invertible.

similar matrix

Def
BBB is similar to AAA: B=M−1AMB=M^{-1}AM\quadB=M−1AM MMM can be any invertible matrix
property

  • BBB and AAA have the same eigenvalues
    By=M−1AMy=λyBy=M^{-1}AMy=\lambda yBy=M−1AMy=λy
    AMy=λMyAMy=\lambda MyAMy=λMy
    x=MyAx=λxx=My\quad Ax=\lambda xx=MyAx=λx
    eigenvector becomes y=M−1xy=M^{-1}xy=M−1x
  • ABABAB and BABABA have the same non-zero eigenvalues
    take M=B−1M=B^{-1}M=B−1 then M−1ABM=(B−1)−1ABB−1=BAM^{-1}ABM=(B^{-1})^{-1}ABB^{-1}=BAM−1ABM=(B−1)−1ABB−1=BA
    ∴BA\therefore BA∴BA is similar to ABABAB
    ∴\therefore∴ they have the same eigenvalues

application
When MATLAB is trying to compute the eigenvalues of AAA. It first takes a bunch of MMM to turn the matrix AAA to a triangular and similar matrix, in which way, the eigenvalues will show up on the diagonal.

symmetric matrix S=STS=S^TS=ST

Def
S=STS=S^TS=ST
property

  • a real symmetric matrix has real eigenvalues while a real anti-symmetric matrix has imaginary eigenvalues
  • eigenvectors are orthogonal
    supposeAu=λ1uAv=λ2vsuppose\quad Au=\lambda_1u\quad Av=\lambda_2vsupposeAu=λ1​uAv=λ2​v
    thenvT(Au)=λ1vTu(vTA)u=(vTAT)u=(Av)Tu=λ2vTu\begin{aligned} then\quad &v^T(Au)=\lambda_1v^Tu\\ &(v^TA)u=(v^TA^T)u=(Av)^Tu=\lambda_2v^Tu \end{aligned}then​vT(Au)=λ1​vTu(vTA)u=(vTAT)u=(Av)Tu=λ2​vTu​
    ∴λ1vTu=λ2vTu\therefore \lambda_1v^Tu=\lambda_2v^Tu∴λ1​vTu=λ2​vTu
    ∴(λ1−λ2)vTu=0\therefore (\lambda_1-\lambda_2)v^Tu=0∴(λ1​−λ2​)vTu=0
    ∴vTu=0\therefore v^Tu=0∴vTu=0
