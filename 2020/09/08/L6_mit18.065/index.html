Lecture6: Singular Value Decomposition (SVD)

MIT 18.065: Matrix Methods in Data Analysis, Signal Processing, and Machine Learning

this note is only a list of the key points

ATAA^TAATA

AAA is m×nm \times nm×n

  • ATAA^TAATA is n×n∴An \times n\quad\therefore An×n∴A is square
  • (ATA)T=ATATT=ATA∴A(A^TA)^T=A^TA^{TT}=A^TA\quad\therefore A(ATA)T=ATATT=ATA∴A is symmetric
  • xTATAx=(Ax)TAx⩾0∴Ax^TA^TAx=(Ax)^TAx\geqslant0\quad\therefore AxTATAx=(Ax)TAx⩾0∴A is positive semidefinite
  • ATA=VΛVTA^TA=V\Lambda V^TATA=VΛVT n×nn\times nn×n with eigenvalues ⩾0\geqslant 0⩾0 and eigenvectors orthogonal
  • AAT=UΛUTAA^T=U\Lambda U^TAAT=UΛUT m×mm\times mm×m with the same eigenvalues as ATA⩾0A^TA\geqslant 0ATA⩾0 (the missing ones are 000s) and eigenvectors orthogonal

SVD: A=UΣVTA=U\Sigma V^TA=UΣVT

Def

for rectangular AAA there is no eigenvalues and eigenvectors but we can find $$Av_i=\sigma_i u_i\quad i=1,2,\dots,r$$big size:

\begin{aligned} &A\begin{bmatrix} \\v_1&v_2&\dots&v_r&v_{r+1}&\dots&v_n\\\\ \end{bmatrix}= \begin{bmatrix} \\u_1&u_2&\dots&u_r&u_{r+1}&\dots&u_m\\\\ \end{bmatrix} \begin{bmatrix} \sigma_1\\ &\sigma_2\\ &&.\\&&&.\\ &&&&\sigma_r\\ &&&&&0 \end{bmatrix}\\ &A_{m\times n}\kern{6em}V_{n\times n}\kern{6em}=\kern{7em}U_{m\times m}\kern{11.5em}\Sigma_{m\times n} \end{aligned} $$small size:

\begin{aligned}
&A\begin{bmatrix}
\v_1&v_2&\dots&v_r\\
\end{bmatrix}=
\begin{bmatrix}
\u_1&u_2&\dots&u_r\\
\end{bmatrix}
\begin{bmatrix}
\sigma_1\
&\sigma_2\
&&.\&&&.\
&&&&\sigma_r
\end{bmatrix}\
&A_{m\times n}\kern{2.3em}V_{n\times r}\kern{2.8em}=\kern{4em}U_{m\times r}\kern{6.7em}\Sigma_{r\times r}
\end{aligned}

$\therefore A=U\Sigma V^{-1}=U\Sigma V^T$ $A^TA=(U\Sigma V^T)^TU\Sigma V^T=V\Sigma^TU^TU\Sigma V^T=V(\Sigma^T\Sigma)V^T$ $AA^T=U\Sigma V^T(U\Sigma V^T)^T=U\Sigma V^TV\Sigma^TU^T=U(\Sigma\Sigma^T)U^T$ $\Sigma$ is diagonal $\therefore \Sigma^T\Sigma=\Sigma\Sigma^T=\begin{bmatrix} \sigma_1^2\\ &\sigma_2^2\\ &&.\\&&&.\\ &&&&\sigma_r^2 \end{bmatrix}$ $u_i$ and $v_i$ are sigular vectors, eigenvectors of $AA^T$ and $A^TA$ $v_i$ are orthonormal bases in the row space $C(A^T)$ $Av_i=\sigma_iu_i$ are the orthonormal bases in the column space $C(A)$ $\sigma_i$ are singular values of $A$, $\sigma_i^2$ are eigenvalues of $A^TA$ and $AA^T$ $det(A)=\displaystyle\prod_{i=0}^{n}\lambda_i=\displaystyle\prod_{i=0}^{r}\sigma_i$ we take $\sigma_1\geqslant\sigma_2\geqslant\dots\geqslant\sigma_r>0$ After $r$, sigular vectors are in the nullspace and the singular value $\sigma=0$ ### calculate sometimes there will be double or triple eigenvalues 1e.g. $\qquad S=\begin{bmatrix} 1&&\\&1&\\&&5 \end{bmatrix}$ $Sx=\lambda x,\quad\lambda_1=\lambda_2=1,\quad\lambda_3=5,\quad x_3=\begin{bmatrix}0\\0\\1\end{bmatrix}$ However, there is a whole plane of $x_1$ and $x_2$ $x_1,x_2=\begin{bmatrix}p\\q\\0\end{bmatrix}$ $p,q$ can be any value we need to pick two that makes $x_1$ and $x_2$ orthogonal so we take $u_i=\dfrac{Av_i}{\sigma_i}$ proof: $suppose\ u_i=\dfrac{Av_i}{\sigma_i}\quad u_j=\dfrac{Av_j}{\sigma_j}\quad i\not=j$ $then\ u_i^Tu_j=(\dfrac{Av_i}{\sigma_i})^T\dfrac{Av_j}{\sigma_j}=\dfrac{v_i^TA^TAv_j}{\sigma_i\sigma_j}=\dfrac{v_i^T\sigma_j^2v_j}{\sigma_i\sigma_j}=\dfrac{\sigma_j}{\sigma_i}v_i^Tv_j=0$ $QED$ ### geometry ![SVD](https://note.youdao.com/yws/api/personal/file/WEBf1050e4634550f41bd69f4b6b2f8c00d?method=getImage&version=18&cstk=l3tPgPOC) ## polar decomposition $A=U\Sigma V^T=(U\Sigma U^T)(UV^T)=SQ$ ## application $\begin{aligned} A&=U\Sigma V^T\\ &=\begin{bmatrix} \\u_1&u_2&\dots&u_r\\\\ \end{bmatrix} \begin{bmatrix} \sigma_1\\ &\sigma_2\\ &&.\\&&&.\\ &&&&\sigma_r \end{bmatrix} \begin{bmatrix} &&v_1^T&&\\ &&v_2^T&&\\ &&\dots&&\\ &&v_r^T&&\\ \end{bmatrix}\\ &= u_1\sigma_1v_1^T+u_2\sigma_2v_2^T+\dots+u_r\sigma_rv_r^T \end{aligned}$ To pull out the most important information in $A$, we take the principal components (i.e. the first terms) out of the SVD of A. # # # # # # # # # # # #
